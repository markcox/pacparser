The PacParser type exposes the following members.
Methods
  NameDescription
Cleanup
This function should be called once you're done with using pacparser engine.
EnableMicrosoftExtensions
Enables a subset of Microsoft PAC extensions - dnsResolveEx, myIpAddressEx, isResolvableEx. These functions are used by Google Chrome and IE to work with IPv6. More info: http://code.google.com/p/pacparser/issues/detail?id=4
FindProxy
Finds proxy for the given URL and Host. This function should be called only after pacparser engine has been initialized (using pacparser_init) and pac script has been parsed (using pacparser_parse_pac_file or pacparser_parse_pac_string).
Initialize
Initializes JavaScript engine and does few basic initializations specific to pacparser.
InitializeMS
Initializes JavaScript engine, Microsoft extensions, and does few basic initializations specific to pacparser.
JustFindProxyFromFile
This function is a wrapper around functions pacparser_init, pacparser_parse_pac_file, pacparser_find_proxy and pacparser_cleanup. If you just want to find out proxy a given set of pac file, url and host, this is the function to call. This function takes care of all the initialization and cleanup.
JustFindProxyFromFileMS
This function is a wrapper around functions pacparser_enable_microsoft_extensions, pacparser_init, pacparser_parse_pac_file, pacparser_find_proxy and pacparser_cleanup. If you just want to find out proxy a given set of pac file, url and host, this is the function to call. This function takes care of all the initialization and cleanup.
JustFindProxyFromString
This function is a wrapper around functions pacparser_init, pacparser_parse_pac_string, pacparser_find_proxy and pacparser_cleanup. If you just want to find out proxy a given set of pac file, url and host, this is the function to call. This function takes care of all the initialization and cleanup.
JustFindProxyFromStringMS
This function is a wrapper around functions pacparser_enable_microsoft_extensions, pacparser_init, pacparser_parse_pac_string, pacparser_find_proxy and pacparser_cleanup. If you just want to find out proxy a given set of pac file, url and host, this is the function to call. This function takes care of all the initialization and cleanup.
ParsePacFile
Reads a PAC file and evaluates it in the JavaScript context created by pacparser_init.
ParsePacString
Reads a PAC string and evaluates it in the JavaScript context created by pacparser_init.
SetIP(IPAddress)
Sets IP address to a custom value. This is the IP address returned by myIpAddress() javascript function.
SetIP(String)
Sets IP address to a custom value. Otherwise, pacparsers uses its default logic to get the host's IP. The IP address is that returned by the myIpAddress() javascript function.
